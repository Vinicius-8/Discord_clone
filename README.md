# Discord_clone
A Discord clone made with Django

## Getting Started

* Clone the repo: 

```bash
$ git clone https://github.com/Vinicius-8/Discord_clone.git

```

* Install dependencies: 

```bash
$ pip install -r requirements.txt

```

* Migrate: 

```bash
$ python manage.py migrate

```

* Run

```bash
$ python manage.py runserver

```
